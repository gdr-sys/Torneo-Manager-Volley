# Torneo Manager Pallavolo

Applicazione web per gestire tornei di pallavolo direttamente dal browser.

Permette di:

- creare gironi
- inserire risultati
- calcolare classifiche automatiche
- generare fasi finali ed eliminatorie
- esportare PDF dei gironi

Tutto funziona in un unico file HTML, senza installazioni o server.

---

# Funzionalità

- Gestione categorie White / Green / Red
- Creazione automatica gironi
- Inserimento risultati partite
- Classifiche aggiornate in tempo reale
- Classifica avulsa
- Quarti, semifinali e finali
- Tema chiaro/scuro
- Salvataggio automatico nel browser
- Esportazione PDF

---

# Come usare

## 1. Apri il file

Apri `index.html` in un browser.

---

## 2. Configura i gironi

- aggiungere gironi
- modificare squadre
- scegliere numero set
- attivare andata/ritorno

---

## 3. Inserisci i risultati

Apri una categoria e salva i punteggi delle partite.

Le classifiche si aggiornano automaticamente.

---

## 4. Crea la fase successiva

Puoi generare:

- nuovi gironi
- eliminatorie
- top 8

---

# Salvataggio

I dati vengono salvati automaticamente nel browser tramite `localStorage`.
